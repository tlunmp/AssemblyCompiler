Instructions

  • ADD (1, ACC = ACC +arg) ----> "ad"
  • BR (1, jump to arg)
  • BRNEG (1, jump to arg if ACC < 0) ----> "brn"
  • BRZNEG (1, jump to arg if ACC <= 0) ----> "brz"
  • BRPOS (1, jump to arg if ACC > 0) ----> "brp"
  • BRZPOS (1, jump to arg if ACC >= 0) ----> "brzp"
  • BRZERO (1, jump to arg if ACC == 0) ----> "brze"
  • COPY (2, arg1arg2) ----> "co"
  • DIV (1, ACC = ACC /arg) ----> "di"
  • MULT (1, ACC = ACC * arg) ----> "mu"
  • READ (1, arg=input integer) ----> "re"
  • WRITE (1, put arg to output as integer) ----> "wr"
  • STOP (0, stop program)
  • STORE (1, arg = ACC) ----> "stor"
  • SUB (1, ACC = ACC - arg) ----> "su"
  • NOOP (0, nothing) ----> "no"
  • LOAD (1, ACC=arg) ----> "lo"
